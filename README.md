# SnapClip
Stealth screenshot-to-clipboard tool for Windows.

## Usage
- Run: `python -m snapclip`
- Hotkey: Alt+Shift+A (arm capture)
- Drag left mouse to select region (blind)
- Release to capture -> clipboard
- Paste anywhere: Ctrl+V
- Cancel: ESC
- Exit: tray icon -> Exit
